## Question3 

Python Code to Validate and Test Solutions to Cesar's Written Exam

To get running: 

- cd into project folder
- $ mkdir virt_env
- $ virtualenv virt_env/virt1 --no-site-packages
- $ source virt_env/virt1/bin/activate
- $ pip install -r requirements.txt

Run: 
$ python question3.py
